# HumanBenchmarkPy

HumanBenchmarkPy is a project for automate/semi-automate all the 
[humanbenchmark](https://humanbenchmark.com)
's tests whit python.

## Requierments

To use these scripts you need to have:

  -Python 3.x
  
  -pyautogui

  -pynput
  
## Installation

```bash
pip install pyautogui pynput
```
